## BC Physical Address Viewer (KML) — Deprecated

The **BC Physical Address Viewer KML application** has been **deprecated** and is no longer maintained or supported.

### Downloading the KML File

If you still need a copy of the KML file, you can download it manually:

1. Open the following link in a new browser tab:  
   <https://github.com/BK01/ols-devkit/blob/gh-pages/pav/BCGov_Physical_Address_Viewer.kml>

2. In the GitHub interface, click **“Download raw file”** to save the KML file to your default download location.
